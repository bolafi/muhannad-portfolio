# Muhannad — Personal Profile Website

A polished, single-page personal profile site for a Qatari professional serving as **Head of Section at Tarsheed** (KAHRAMAA's National Program for Conservation and Energy Efficiency).

## Stack

- **Next.js 15** (App Router) + **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** for subtle, executive-grade motion
- **lucide-react** for crisp icons

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Editing content

All copy, names, and placeholder data live in `src/content/site.ts`. Update the values there to change every section without touching components.

## Build

```bash
npm run build
npm start
```
